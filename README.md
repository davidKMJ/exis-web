# exis-web
![Static Badge](https://img.shields.io/badge/status-in_progress-orange?style=for-the-badge)
![Static Badge](https://img.shields.io/badge/type-side_project-blue?style=for-the-badge)

A web prototype for EXIS, an AI-powered fitness platform that provides personalized workout plans, AI trainer guidance, community support, and team collaboration. Built with React Router v7, Supabase.

## How to Start

### Environment
- Node.js 18+
- npm
- Git

### Quick Start

```bash
# Clone the repository
git clone https://github.com/davidKMJ/exis-web.git
cd exis-web

# Install dependencies
npm install

# Set up environment variables
# Create a .env file with the following variables:
# SUPABASE_URL=your_supabase_url
# SUPABASE_ANON_KEY=your_supabase_anon_key
# SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
# DATABASE_URL=your_database_url
# API_URL=your_backend_api_url

# Run the application
npm run dev

# Open in browser
# Navigate to http://localhost:5173
```

## Key Features
1. **AI-Powered Workouts** – Personalized daily workout plans generated by AI based on your fitness profile and goals
2. **AI Trainer** – Chat with an intelligent fitness trainer that creates your workout profile and provides guidance
3. **Workout Tracking** – Track exercises, sets, reps, and progress with real-time feedback
4. **Community Discussions** – Engage in fitness-related posts, share tips, and connect with other fitness enthusiasts
5. **Workout Teams** – Find workout partners and join fitness groups to stay motivated together
6. **User Dashboard** – View your progress, manage your profile, and access personalized analytics

## Technical Stack
- **React Router v7** – Type-safe routing and server-side rendering
- **TypeScript** – Static type checking for enhanced developer experience
- **Supabase** – Backend-as-a-service for authentication, database, and real-time features
- **Drizzle ORM** – Type-safe database queries with PostgreSQL
- **Tailwind CSS & Shadcn/ui** – Utility-first styling with accessible component library
- **Magic UI** – Beautiful, animated UI components for modern interfaces
- **OpenAI API** – Powers the AI trainer and workout planning features
- **Luxon** – Date and time manipulation for workout scheduling
